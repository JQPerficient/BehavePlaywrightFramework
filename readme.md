# BehaveBDDPlaywrightAI
    A **BDD test automation framework (Python)** that combines **Behave**, **API**, 
    **Playwright* and **Generative AI** (Google Gemini) to deliver intelligent, 
    self-healing and visually-aware end-to-end testing.
